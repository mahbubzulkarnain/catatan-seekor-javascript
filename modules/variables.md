### VAR ( Global )
```javascript
var iniVariableGlobal = TYPE_DATA;
```

### LET ( Local )
```javascript
let iniVariableGlobal = TYPE_DATA;
```

### CONST ( Read Only )
```javascript
let iniVariableGlobal = TYPE_DATA;
```

### Assignment Operator
```javascript
a  = 1;              // a = 1;
a += 1;              // a = a + 1;
a -= 1;              // a = a - 1;
a *= 1;              // a = a * 1;
a /= 1;              // a = a / 1;
a %= 1;              // a = a % 1;
```
```javascript
a  = 1 + 1;          //  1  + 1 = 2
b  = a + 1;          // (2) + 1 = 3
c  = a = b = 3;      // c = 3, a = 3, b = 3
d  = a += b /= c;    // ? 
```
